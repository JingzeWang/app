CodeForces-1038D: Slime

链接：https://codeforces.com/problemset/problem/1038/D

$Time\;Limit:\;2000\;ms$

$Memory\;Limit:\;262144\;kB​$

Description

There are $n​$ slimes in a row. Each slime has an integer value (possibly negative or zero) associated with it.

Any slime can eat its adjacent slime (the closest slime to its left or to its right, assuming that this slime exists).

When a slime with a value $x​$ eats a slime with a value $y​$, the eaten slime disappears, and the value of the remaining slime changes to $x−y​$.

The slimes will eat each other until there is only one slime left.

Find the maximum possible value of the last slime.

Input

The first line of the input contains an integer $n​$($1≤n≤500000​$) denoting the number of slimes.

The next line contains $n$ integers $a_i$ ($−10^9≤a_i≤10^9$), where $a_i$ is the value of $i$-th slime.

Output

Print an only integer — the maximum possible value of the last slime.

Examples

Output

Output

Note

In the first example, a possible way of getting the last slime with value $4$ is:

  • Second slime eats the third slime, the row now contains slimes $2,−1,1$
  • Second slime eats the third slime, the row now contains slimes $2,−2$
  • First slime eats the second slime, the row now contains $4$

In the second example, the first slime can keep eating slimes to its right to end up with a value of $4​$.

题解

水题，三种情况

  • 有正有负的情况直接将所有数的绝对值相加
  • 全正或全负的情况则将所有数的绝对值相加后减去最小的绝对值
  • $n=1$需要特判

You have an array $a$ consisting of $n$ integers. Each integer from 1 to $n$ appears exactly once in this array.

For some indices $i​$ (1 ≤ $i​$ ≤ $n​$ - 1) it is possible to swap $i​$-th element with ($i​$ + 1)-th, for other indices it is not possible. You may perform any number of swapping operations any order. There is no limit on the number of times you swap $i​$-th element with ($i​$ + 1)-th (if the position is not forbidden).

一、一阶偏导数

1. 一阶偏导数的定义和计算

Def 1 一阶偏导数

$$
若\lim_{\Delta x\rightarrow0}\frac{f(x_0+\Delta x,y_0)-f(x_0,y_0)}{\Delta x}存在,\\
则称此极限值为z=f(x,y)在点P_0(x_0,y_0)关于x的偏导数,\
记为\frac{\partial z}{\partial x}|_{x=x_0\y=y_0}或\frac{\partial f}{\partial x}|_{x=x_0\y=y_0}或z_{x}(x_0,y_0)或f_{x}(x_0,y_0)\
即f_{x}(x_0,y_0)=\lim_{\Delta x\rightarrow0}\frac{f(x_0+\Delta x,y_0)-f(x_0,y_0)}{\Delta x}\
类似可定义\lim_{\Delta y\rightarrow0}\frac{f(x_0,y_0+\Delta y)-f(x_0,y_0)}{\Delta y}(存在)\
称为z=f(x,y)在点P_0(x_0,y_0)关于y的偏导数\
记为\frac{\partial z}{\partial y}|_{x=x_0\y=y_0}或\frac{\partial f}{\partial y}|_{x=x_0\y=y_0}或z_{y}(x_0,y_0)或f_{y}(x_0,y_0)\
\color{red}{二者合称一阶偏导数}
$$

Def 2 偏导函数

$$
z=f(x,y)\quad(x,y)\in D\
\begin{cases}
f_x(x,y),z_x(x,y),\frac{\partial f}{\partial x},\frac{\partial z}{\partial x}&z=f(x,y)对x的偏导函数\
f_y(x,y),z_y(x,y),\frac{\partial f}{\partial y},\frac{\partial z}{\partial y}&z=f(x,y)对y的偏导函数\
\end{cases}
$$

Remark

  1. $$
    f_{x}(x_0,y_0)=f_x(x,y)|_{x=x_0\y=y_0}\
    f_{y}(x_0,y_0)=f_y(x,y)|_{x=x_0\y=y_0}\
    f_{x}(x,y)=\lim_{\Delta x\rightarrow0}\frac{f(x+\Delta x,y)-f(x,y)}{\Delta x}
    $$

  2. $$
    偏导函数\stackrel{简称}{\longrightarrow}偏导数\
    导函数\stackrel{简称}{\longrightarrow}导数\
    $$

  3. $$
    求z=f(x,y)的偏导数\stackrel{本质}{\longrightarrow}求一元函数的导数\
    \begin{cases}
    求\frac{\partial z}{\partial x}-视y为常数\
    求\frac{\partial z}{\partial y}-视x为常数
    \end{cases}
    $$

  4. e.g.1

    f(x,y)=xy+x^2+y^2,求f_x(0,1),f_y(2,0)\
    \begin{split}
    解:&f_x(x,y)=y+2x \Rightarrow f_x(0,1)=1\
    &f_y(x,y)=x+2y \Rightarrow f_y(2,0)=2
    \end{split}
    $$

    解:&z_x=2x\sin{xy}+x^2y\cos{xy}\
    &z_y=x^3\cos{xy}
    \end{split}
    $$

3. 偏导存在与连续性的关系

$$
\begin{split}
&一元函数y=f(x)在点x_0处\quad&可导\frac{\color{red}\rightharpoonup}{\color{red}\not\leftharpoondown}连续
&\quad\&&e.g.f(x)=|x|在点x_0=x处\
&二元函数y=f(x,y)在点(x_0,y_0)处\quad&偏导存在{\color{red}\not\rightleftharpoons}连续
\end{split}
$$

e.g.1 $连续\nRightarrow 偏导存在​$
$$
z=|x|+|y|在点(0,0)处,z=f(x,y)=|x|+|y|\rightarrow 0
(\begin{split}x\rightarrow0\y\rightarrow0\end{split})=f(0,0)\
\begin{split}
1.&z=f(x,y)=|x|+|y|在点(0,0)连续\
2.&z_x(0,0)=\lim_{\Delta x\rightarrow 0}\frac{f(0+\Delta x,0)-f(0,0)}{\Delta x}=\lim_{\Delta x\rightarrow0}\frac{|\Delta x|}{\Delta x}不存在\
\Rightarrow&z=f(x,y)=|x|+|y|在点(0,0)关于x的偏导不存在\
同理可证&z=f(x,y)=|x|+|y|在点(0,0)关于y的偏导不存在\
\end{split}
$$
e.g.2 $偏导存在\nRightarrow 连续$
$$
z=f(x,y)=\begin{cases}\frac{xy}{x^2+y^2}&(x,y)\ne(0,0)\0&(x,y)=(0,0)\end{cases}\
\begin{split}
在(0,0)点,有&z_x(0,0)\stackrel{用定义}{=}z_x(0,0)=\lim_{\Delta x\rightarrow 0}\frac{f(0+\Delta x,0)-f(0,0)}{\Delta x}=\lim_{\Delta x\rightarrow 0}\frac{0-0}{\Delta x}=0\
同理，可得&z_y(0,0)=0,偏导存在\
&\lim_{x\rightarrow0\y\rightarrow0}\frac{xy}{x^2+y^2}不存在(“路径法”)\Rightarrow z=f(x,y)在(0,0)点不连续
\end{split}
$$

定理

$$
z=f(x,y)在点(x_0,y_0)的某邻域内处处存在偏导数f_x(x,y),f_y(x,y)且各偏导数在此邻域内有界\
\Rightarrow f(x,y)在(x_0,y_0)连续
$$

二、高阶偏导数

Def 3

$$
\left.\begin{split}
\frac{\partial^2 z}{\partial x^2}=f_{xx}(x,y)=\frac{\partial}{\partial x}(\frac{\partial f}{\partial x})=\frac{\partial^2 f}{\partial x^2}\
\frac{\partial^2 z}{\partial y^2}=f_{yy}(x,y)=\frac{\partial}{\partial y}(\frac{\partial f}{\partial y})=\frac{\partial^2 f}{\partial y^2}\
\left.\begin{split}
\frac{\partial^2 z}{\partial x\partial y}=f_{xy}(x,y)=\frac{\partial}{\partial y}(\frac{\partial f}{\partial x})\
\frac{\partial^2 z}{\partial y\partial x}=f_{yx}(x,y)=\frac{\partial}{\partial x}(\frac{\partial f}{\partial y})
\end{split}\right}混合偏导
\end{split}\right}
\begin{split}
&z=f(x,y)的二阶偏导数(4个),可推广至二阶以上\
&\frac{\partial^3 z}{\partial x^3},\frac{\partial^3 z}{\partial y^3},\cdots\frac{\partial^3 z}{\partial y\partial x\partial y},\frac{\partial^3 z}{\partial y^2\partial x}\
&C^1_2C^1_2C^1_2=8\
&三阶偏导(8种)
\end{split}
$$

e.g.1
$$
z=xe^x\sin y,求四个二阶偏导数\
\left{\begin{split}

&\frac{\partial^2 z}{\partial y^2}\
&\left.\begin{split}
\frac{\partial^2 z}{\partial x\partial y}\
\frac{\partial^2 z}{\partial y\partial x}
\end{split}\right}=e^x(1+x)\cos y
\left{\begin{split}
&\frac{\partial z}{\partial x}=(e^x+xe^x)\sin y\
&\frac{\partial z}{\partial y}=xe^x\sin y
\end{split}\right.
\end{split}\right.
$$
e.g.2
$$
f(x,y)=
\begin{cases}
xy\frac{x^2-y^2}{x^2+y^2}&(x,y)\ne(0,0)\
0&(x,y)=(0,0)
\end{cases},求f_{xy}(0,0),f_{yx}(0,0)\
{\color{red}\bigstar} f_{xy}(0,0)\stackrel{定义}{=}\lim_{\Delta y\rightarrow 0}\frac{f_x(0,0+\Delta y)-f_x(0,0)}{\Delta y}\
\left.\begin{split}
(x,y)\ne (0,0)时,f_x(x,y)\stackrel{公式}{=}\frac{(3x^2-y^3)(x^2+y^2)-2x\cdot xy(x^2-y^2)}{(x^2+y^2)^2}\
(x,y)=(0,0)时,f_x(x,y)\stackrel{定义}{=}\lim_{\Delta x\rightarrow 0}\frac{f(0+\Delta x,0)-f(0,0)}{\Delta x}=\lim_{\Delta x\rightarrow 0}\frac{0-0}{\Delta x}=0
\end{split}\right}\\Rightarrow f_{xy}(0,0)=\lim_{\Delta y\rightarrow 0}\frac{\frac{-(\Delta y)^5}{(\Delta y)^4}}{\Delta y}=-1\
类似地,f_{yx}(0,0)=1
$$

定理

混合偏导与求导次序无关

Remark 可推广

e.g.1
$$
验证:z=\ln\sqrt{x^2+y^2}满足二维Laplance方程\frac{\partial^2 z}{\partial x^2}+\frac{\partial^2 z}{\partial y^2}=0\
\begin{split}
证明:&\frac{\partial z}{\partial x}=\frac{1}{\sqrt{x^2+y^2}}\cdot\frac{x}{\sqrt{x^2+y^2}}=\frac{x}{x^2+y^2}\
&\frac{\partial^2 z}{\partial x^2}=\frac{x^2+y^2-2x^2}{(x^2+y^2)^2}=\frac{y^2-x^2}{(x^2+y^2)^2}\

\end{split}
$$

四、多元函数的连续性

Def 4

$$
\lim_{x\rightarrow x_0\y\rightarrow y_0}f(x,y)=f(x_0,y_0)\Rightarrow z=f(x,y)在P_0(x_0,y_0)连续\
否则,称z=f(x,y)在P_0(x_0,y_0)不连续,P_0为间断点
\begin{cases}
1.P_0点无定义\
2.极限不存在\
3.极限存在但\ne f(P_0)
\end{cases}
$$

例如
$$
f(x,y)=
\begin{cases}
(x^2+y^2)\sin\frac{1}{x^2+y^2}&(x,y)\ne (0,0)\
0&(x,y)= (0,0)
\end{cases}\
\lim_{x\rightarrow x_0\y\rightarrow y_0}f(x,y)=0=f(0,0),函数连续
$$
$Remark$ 可推广

Def 5

$$
z=f(x,y)在D上的每一点都连续，称f(x,y)在D上连续或f(x,y)为D上的连续函数
$$

Def 6

$$
多元初等函数:由常数及一元基本初等函数经过有限次四则运算和(或)有限次复合形成的函数
$$

e.g.
$$
\lim_{x\rightarrow x_0\y\rightarrow y_0}\frac{\ln(2^x+e^y)+1}{\sin(x^2+y^2)}=\frac{\ln3+1}{\sin1}\
\color{red}{一切二元初等函数在其定义域内都是连续的}\
z=f(x,y)\quad(x,y)\in D\
\lim_{x\rightarrow x_0\y\rightarrow y_0}f(x,y)=f(x_0,y_0)为二元初等函数的极限\color{red}{(x_0,y_0)}\in D\
$$
推广
$$
\begin{split}
有界闭区域上二元连续函数的性质\
闭区间上一元连续函数的性质
\end{split}
}整体性质
\begin{cases}
1.有界性\
2.最值性\
3.介值性\
4.一致连续性\
\end{cases}
$$

给出$1-n$的两个排列$P1$和$P2$，求它们的最长公共子序列。

输入格式

第一行是一个数$n$，

接下来两行，每行为$n$个数，为自然数$1-n$的一个排列。

输出格式

一个数，即最长公共子序列的长度

输入样例#1

输出样例#1

说明

【数据规模】

对于$50\%​$的数据，$n\le 1000​$

对于$100\%$的数据，$n\le100000​$

题解

将$P1$中的数换成$1-n$，用数组记录下对应的替换

然后将$P2$中的数依$P1$的数字对应关系变换

易知变换后$P2$各位的值即其在$P1$中的位置

那么题目变成了求$P2$最大上升子序列长度的问题

拿个$O(nlog_2n)$的板子套一下即可

一、平面点集的一些概念

一元函数 值域 $R$-直线

二元函数 值域 $R^2$-平面

1. 邻域

1. 定义

$P_0(x_0,y_0)为平面上的一点,\delta>0​$

$点P的\delta邻域U(P_0,\delta)={P||PP_0|<\delta|}​$

$其中|PP_0|=\sqrt{(x-x_0)^2+(y-y_0)^2}$ $P(x,y),P_0(x_0,y_0)$

2. 几何意义

3. 简记

$U(P_0)$

4. 点$P_0$的去心邻域

$\mathring{U}(P_0,\delta)={P|0<|PP_0|<\delta},简记为\mathring{U}(P_0),几何意义略$

2. 区域

$E-平面点集$

1. 内点

$$
\begin{array}
&i) &P\in E\
ii) &\exists U(P)\subset E\
\end{array}
\Rightarrow P为E的内点
$$

2. 外点

$$
\begin{array}
&i) &P\notin E\
ii) &\exists U(P)\cap E = \varnothing\
\end{array}
\Rightarrow P为E的外点
$$

3. 边界点

$$
\begin{array}
&i) &P\in E\;or\;P\notin E\
ii) &\forall U(P):
\begin{split}
&U(P)\cap E \ne \varnothing\
&U(P)\cap R^2\backslash E \ne \varnothing\
\end{split}\
\end{array}
\Rightarrow P为E的边界点
$$

$$
E的边界点的全体-E的边界,记为\partial E
$$

Remark

$给定一平面点集E,R^2上的点或为内点,或为外点,或为边界点$

<分类>三者必居其一,且只居其一​

4. 开集

若$E​$上的点均为内点,则称$E​$为开集

5. 联通的

若开集$D$内的任何一点可以用折线连接,且折线上的点均属于$P$,则称$D$为连通的开集

6. 开区域

$联通的+开集=开区域$(开区间的推广)

7. 闭区域

$开区域+边界=闭区域​$

8. 有界点集

$O(0,0),\exists k>0,\forall p \in E,|OP| \le k,则称E为有界点集$

e.g.

$$
E_1={(x,y)|1<x^2+y^2<4}:开区域\
E_2={(x,y)|1\le x^2+y^2\le 4}:闭区域\
E_3={(x,y)|x^2+y^2\ge 0}:无界闭区域
$$

Remark

$$
\begin{array}{c|l|l}
{}&{值域}&{点}\
\hline
{一元函数}&{一维直线上的点集R}&{x}\
\hline
{二元函数}&{二维平面上的点集R^2}&{(x,y)}\
\hline
{三元函数}&{三维空间上的点集R^3}&{(x,y,z)}\
\hline
{\cdots}&{\cdots\cdots}&{\cdots}\
\hline
{n元函数}&{n维空间上的点集R^n}&{(x_1,x_2,x_3,\cdots,x_n)}\
\end{array}
\
n维空间可类似定义
$$

二、多元函数的概念

e.g.1 $S_\Delta=\frac {1}{2}dh$

e.g.2 理想气体压强$P=R\frac{I}{U}$

Def 1

$D-平面点集,R-实数集​$

$若对\forall P(x,y)\in D,按照对应法则f,有唯一确定的z\in R与之对应,则称z为x,y的二元函数,记为$
$$
z=f(x,y)\cdots(x,y)\in D
$$

$$
\begin{cases}
x,y&-自变量\
z&-因变量\
D&-定义域\
{z|z=f(x,y),(x,y)\in R}&-值域\
\end{cases}
$$

Remark

1. 类似可定义

三元函数$U=f(x,y,z)\cdots(x,y,z)\in D$

$n元函数U=f(x_1,x_2,\cdots,x_n)\cdots (x_1,x_2,\cdots,x_n)\in D$

2. 自然定义域

e.g.

$z=\tan(x^2+y^2)\rightarrow D={(x,y)|x^2+y^2\ne k\pi +\frac{\pi}{2},k\in Z}$

$z=\arcsin(x^2+y^2)\rightarrow D={(x,y)|x^2+y^2\le 1}​$

$z=\ln(x+y) \rightarrow D={(x,y)|x+y>0}$

统一<点函数观点>

$U=f(P),P\in D一点,P(x_1,x_2,\cdots,x_n)​$
$$
\begin{cases}
n=1&一元函数\
n\ge2&多元函数
\end{cases}
$$

Def 2

$设Z=f(x,y)\cdots(x,y)\in D,$

$则空间点集{(x,y,z)|z=f(x,y),(x,y)\in D}为二元函数z=f(x,y)\cdots (x,y)\in D的图形$

Remark

二元函数的图形通常是一张曲面

e.g.
$$
\begin{array}
&z=\sqrt{a^2-x^2-y^2}&上半球面\
z=xy&马鞍面(双曲抛物面)\
\end{array}
$$

三、多元函数的极限(二元函数)

$p\rightarrow p_0(任何方式)$

$f(p)\rightarrow A(常数)$

Def 3$(\varepsilon-\delta)$

$f(x,y)在区域D(或开或闭)上有定义,P_0(x_0,y_0)为D的内点或非孤立的边界点(聚点)$

$若\exists A\in R,对\forall \varepsilon >0,\exists \delta >0,\forall P(x,y)\in D$

$0<|PP_1|=\sqrt{(x-x_0)^2+(y-y_0)^2}<\delta(即P\in \mathring{U}(P_0,\delta)),有|f(x,y)-A|<\varepsilon​$

$则称A为f(x,y)在点(x,y)趋于P(x_0,y_0)时的极限,记为$
$$
\lim_{x\rightarrow x_0\y \rightarrow y_0}f(x,y)=A\
\lim_{(x,y)\rightarrow(x_0,y_0)}f(x,y)=A\
f(x,y)\rightarrow A(\rho\rightarrow0),其中\rho=|PP_0|\
\lim_{p \rightarrow p_0}f(p)=A
$$

Remark

1. 二元函数的极限-(二重极限)

$(\begin{array} &x\rightarrow x_0\ y \rightarrow y_0\end{array})同时进行​$

e.g.
$$
证明:\lim_{x\rightarrow x_0\y \rightarrow y_0}^{}(x^2+y^2)\sin \frac{1}{x^2+y^2}=0
$$

$$
(\varepsilon-\delta)记\forall\varepsilon>0,\exists \delta=\sqrt{\varepsilon}>0\
\forall (x,y):0<\sqrt{(x-0)^2+(y-0)^2}<\delta时\
有|(x^2+y^2)\sin\frac{1}{x^2+y^2}-0|\le x^2+y^2 < \varepsilon,得证
$$

2. $P(x,y)$极限不存在

$$
\begin{array}
&1.P\rightarrow P_0(某种方式):f(x,y)极限不存在\
2.P\rightarrow P_0(两种不同方式):f(x,y)极限不同(“路径法”)\
\end{array}
\Rightarrow 极限不存在
$$

FZU-2020: 组合

链接：http://acm.fzu.edu.cn/problem.php?pid=2020

$Time\;Limit:\;1000\;ms$

$Memory\;Limit:\;37268\;kB​$

Description

给出组合数$C(n,m)​$, 表示从$n​$个元素中选出$m​$个元素的方案数。例如$C(5,2)=10​$, $C(4,2) = 6​$.可是当$n,m​$比较大的时候，$C(n,m)​$很大！于是$xiaobo​$希望你输出 $C(n,m)\;mod\;p​$的值！

Input

输入数据第一行是一个正整数T,表示数据组数$ (T \le 100)​$ 接下来是T组数据，每组数据有$3​$个正整数 $n, m, p (1 \le m \le n \le 10^9, m \le 10^4, m < p < 10^9, p是素数)​$

Output

对于每组数据，输出一个正整数，表示$C(n,m)\;mod\;p​$的结果。

Sample Input

Sample Output

题解

$Lucas$定理

$$
\begin{pmatrix}
m\n
\end{pmatrix}
= \prod^k_{i=0}
\begin{pmatrix}
m_i\n_i
\end{pmatrix}(mod\;p)
\tag{1}
$$

$$
m=m_kp^k+m_{k-1}^p{k-1}+\cdots+m_1p+m_0
\tag{2}
$$

$$
n=n_kp^k+n_{k-1}^p{k-1}+\cdots+n_1p+n_0
\tag{3}
$$

则有
$$
\begin{pmatrix}
p\n
\end{pmatrix}
=\frac
{p\cdot(p-1)\cdots(p-n+1)}
{n\cdot(n-1)\cdots1}
\tag{4}
$$

可得
$$
ans= \prod^n_{i=0}\frac{(m-n+i)}{i}
\tag{5}
$$
又由于取模不能进行除法，则需要求逆元，即
$$
ans= \prod^n_{i=0}(m-n+i)\cdot inv(i)
\tag{6}
$$
由费马小定理
$$
a^{p-1}\equiv 1(mod\; p)
\tag{7}
$$
则有
$$
a^{p-1}=a\cdot a^{p-2}=1(mod\;p)
\tag{8}
$$
则
$$
inv(a)=a^{p-2}\tag{9}
$$
然后求解即可
